- 웹 페이지의 디자인과 레이아웃을 구성하는 언어

- 내부(Internal) 스타일
- 외부(External) 스타일

# CSS 기본 구조와 문법
- 선택자(Seletor)
    - '누구를' 꾸밀지 지정하는 부분
- 선언(Declaration)
    - '어떻게' 꾸밀지에 대한 구체적인 한 줄의 명령
    - 속성과 값이 한 쌍으로 이루어지며, 세미콜론(;)으로 끝남
- 속성(Property)
    - 바꾸고 싶은 스타일의 종류를 나타냄
- 값(Value)
    - 속성에 적용할 구체적인 설정을 나타냄
